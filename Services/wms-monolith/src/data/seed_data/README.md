# Seed Data

This folder contains test data for development and testing purposes.

## Files Structure

- `users.json` - Test user accounts
- `warehouses.json` - Sample warehouse data
- `products.json` - Sample product catalog
- `customers.json` - Sample customer data
- `inventory.json` - Initial inventory data

## Usage

These seed files can be used to:
- Populate development database
- Run integration tests
- Demonstrate system functionality
- Performance testing

## Data Format

All seed files use JSON format for easy parsing and modification.

## Important Notes

- **DO NOT** use production data in seed files
- All passwords are test passwords only
- Email addresses are fictional
- Product data is for testing purposes only
